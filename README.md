# setu

基于python-mirai的色图插件

## 部署

前略，部署mirai并启用mirai-api-http插件
'''
git clone https://github.com/umauc/setu.git
cd setu
pip3 install -r requirements.txt
cp config.example.py config.py # 之后修改config.py中的参数
python3 setu.py
'''
